# AGENTS

## Agent Focus Areas

- Preserve package public API stability.
- Keep quality gates green: lint, static analysis, tests, coverage.
- Maintain docs structure aligned with DTO index layout.
- Do not remove client-specific runtime assets (`config/`, `scripts/`, `phpbench.json`, Docker files).
