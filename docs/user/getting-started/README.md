## What is JOOClient?

JOOClient is an enhanced Guzzle HTTP client wrapper designed specifically for Laravel 12. It provides:

- ✅ **Intelligent Logging** - Auto-detect and log to MySQL, MongoDB, or Monolog
- ✅ **Caching** - Redis and Filesystem caching support
- ✅ **Retry Logic** - Automatic retry with exponential backoff
- ✅ **Rate Limiting** - Prevent 429 errors
- ✅ **Circuit Breaker** - Resilience patterns
- ✅ **Request Chaining** - Sequential requests with conditionals
- ✅ **And much more!**
