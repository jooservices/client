# Deployment

Production deployment guides and best practices.

## Contents

- **[Production Deployment](production.md)** - Production deployment guide (standard PHP hosting)

## Quick Checklist

- [ ] Environment variables configured
- [ ] SSL verification enabled
- [ ] Batch logging enabled for high traffic
- [ ] Redis cache configured (if using caching)
- [ ] Log rotation scheduled
- [ ] Health checks configured
- [ ] Monitoring set up
- [ ] No Docker dependency in deployment path
